# 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 子串和子序列混淆 | 子串必须连续 |
| dp 下标混乱 | 字符访问 i-1 |
| 初始化漏空串 | 第 0 行列单独处理 |

---

# 面试速记

- 两个序列大概率二维 DP。
- 相等通常看左上角。
- 不等通常丢一个字符。
