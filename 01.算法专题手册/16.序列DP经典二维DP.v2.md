# 序列 DP：经典二维 DP

## 本章目标

训练两个字符串 / 两个序列题的第一反应：看到匹配、删除、编辑、公共子序列，优先想到二维 DP。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 两个字符串比较 | 二维 DP |
| 最长公共子序列 | LCS |
| 最少编辑次数 | 编辑距离 |
| 子序列匹配个数 | `dp[i][j]` |
| 回文子序列 | 区间 DP / 二维 DP |
| 连续公共部分 | 最长公共子数组 |

## 高频模板 1：最长公共子序列

### 秒判关键词

两个字符串、公共子序列、不要求连续。

### 母题

LC1143 Longest Common Subsequence

### 标准代码

```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[m][n];
}
```

### 一句话记忆

字符相等就一起用，不等就丢掉一边试。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Delete Operation for Two Strings | 用 LCS 算删除次数 |
| Uncrossed Lines | 本质就是 LCS |
| Shortest Common Supersequence | 基于 LCS 还原路径 |

### 常见坑点

- `dp[i][j]` 表示前 i 个和前 j 个字符。
- 字符访问是 `charAt(i - 1)`。

## 高频模板 2：编辑距离

### 秒判关键词

插入、删除、替换、最少操作。

### 母题

LC72 Edit Distance

### 标准代码

```java
public int minDistance(String word1, String word2) {
    int m = word1.length(), n = word2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 0; i <= m; i++) dp[i][0] = i;
    for (int j = 0; j <= n; j++) dp[0][j] = j;

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1];
            } else {
                dp[i][j] = 1 + Math.min(dp[i - 1][j - 1],
                    Math.min(dp[i - 1][j], dp[i][j - 1]));
            }
        }
    }
    return dp[m][n];
}
```

### 一句话记忆

不相等时，在替换、删除、插入里选最小。

### 高频变体

- Delete Operation：只允许删除。
- One Edit Distance：判断是否只差一步，可双指针。

### 常见坑点

- 空串变成长度 j 的串，需要 j 次插入。
- 三个操作对应三个来源，不要漏。

## 高频模板 3：子序列计数

### 秒判关键词

一个字符串中有多少个目标子序列。

### 母题

LC115 Distinct Subsequences

### 标准代码

```java
public int numDistinct(String s, String t) {
    int m = s.length(), n = t.length();
    long[][] dp = new long[m + 1][n + 1];
    for (int i = 0; i <= m; i++) dp[i][0] = 1;

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            dp[i][j] = dp[i - 1][j];
            if (s.charAt(i - 1) == t.charAt(j - 1)) {
                dp[i][j] += dp[i - 1][j - 1];
            }
        }
    }
    return (int) dp[m][n];
}
```

### 一句话记忆

当前字符可用也可不用，不等时只能不用。

### 高频变体

- Is Subsequence：简单双指针即可。
- Number of Matching Subsequences：多查询时分桶。

### 常见坑点

- 空目标串永远有 1 种匹配方式。
- 计数可能很大，注意类型。

## 高频模板 4：回文子序列

### 秒判关键词

最长回文子序列、从两端收缩。

### 母题

LC516 Longest Palindromic Subsequence

### 标准代码

```java
public int longestPalindromeSubseq(String s) {
    int n = s.length();
    int[][] dp = new int[n][n];

    for (int i = n - 1; i >= 0; i--) {
        dp[i][i] = 1;
        for (int j = i + 1; j < n; j++) {
            if (s.charAt(i) == s.charAt(j)) {
                dp[i][j] = dp[i + 1][j - 1] + 2;
            } else {
                dp[i][j] = Math.max(dp[i + 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[0][n - 1];
}
```

### 一句话记忆

两端相等就一起用，不等就丢一端。

### 高频变体

- Longest Palindromic Substring：连续回文，更常用中心扩展。
- Palindromic Substrings：中心扩展优先。

### 常见坑点

- 子序列不要求连续，子串要求连续。
- 区间 DP 通常 i 从后往前。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| `i` 和字符下标混乱 | `dp` 用长度，字符用 `i-1` |
| 子序列和子串混淆 | 子序列可跳，子串连续 |
| 二维 DP 初始化漏空串 | 第 0 行 / 第 0 列先写 |
| 编辑距离漏操作 | 替换、删除、插入三选一 |
| 回文区间遍历方向错 | i 从右往左 |

## 一题多变体

```text
Longest Common Subsequence
↓
Delete Operation
↓
Uncrossed Lines
```

```text
Edit Distance
↓
Delete Operation
↓
One Edit Distance
```

```text
Longest Palindromic Subsequence
↓
Palindromic Substrings
```

## 面试速记

- 两个序列：大概率二维 DP。
- `dp[i][j]`：前 i 个和前 j 个。
- 相等：通常看 `dp[i-1][j-1]`。
- 不等：通常丢一个字符。
- 子串连续，子序列可跳。
