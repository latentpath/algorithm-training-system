# 动态规划基础：线性 DP

## 本章目标

训练一维 DP 的第一反应：看到“到当前位置的最优值 / 方案数”，先想状态和转移。

---

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 爬楼梯 / 斐波那契 | 一维 DP |
| 到第 i 个位置的最优值 | `dp[i]` |
| 不能选相邻 | 打家劫舍 |
| 最大连续子数组和 | Kadane |
| 按一位 / 两位解释 | Decode Ways |
| 问所有方案枚举 | 回溯优先，不是 DP 优先 |

---

## 高频模板 1：基础一维 DP

### 秒判关键词

到第 i 步、依赖前几个位置、方案数。

### 母题

LC70 Climbing Stairs

### 标准代码

```java
public int climbStairs(int n) {
    if (n <= 2) return n;

    int prev2 = 1;
    int prev1 = 2;

    for (int i = 3; i <= n; i++) {
        int cur = prev1 + prev2;
        prev2 = prev1;
        prev1 = cur;
    }

    return prev1;
}
```

### 一句话记忆

当前答案来自前一步和前两步。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Min Cost Climbing Stairs | 当前取前两个最小代价 |
| Fibonacci Number | 同样滚动变量 |
| Decode Ways | 按一位 / 两位转移 |

### 常见坑点

- 先写清楚 `dp[0]`、`dp[1]`。
- 滚动变量只是优化，想不清先写数组。

## 高频模板 2：打家劫舍

### 秒判关键词

不能选相邻、选或不选、最大金额。

### 母题

LC198 House Robber

### 标准代码

```java
public int rob(int[] nums) {
    int skip = 0;
    int take = 0;

    for (int x : nums) {
        int newTake = skip + x;
        int newSkip = Math.max(skip, take);

        take = newTake;
        skip = newSkip;
    }

    return Math.max(skip, take);
}
```

### 一句话记忆

选当前，就不能选上一个；不选当前，就继承上一个最好。

### 高频变体

| 题目 | 变化点 |
|---|---|
| House Robber II | 环形数组，拆成两段 |
| Delete and Earn | 按值聚合后变打家劫舍 |
| House Robber III | 树上选点，转树形 DP |

### 常见坑点

- 环形不能同时选头尾。
- 状态含义要固定：`take` 是选当前，`skip` 是不选当前。

## 高频模板 3：最大连续子数组和

### 秒判关键词

连续子数组、最大和、可以从当前重新开始。

### 母题

LC53 Maximum Subarray

### 标准代码

```java
public int maxSubArray(int[] nums) {
    int cur = nums[0];
    int ans = nums[0];

    for (int i = 1; i < nums.length; i++) {
        cur = Math.max(nums[i], cur + nums[i]);
        ans = Math.max(ans, cur);
    }

    return ans;
}
```

### 一句话记忆

前面贡献为负就丢掉，从当前重新开始。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Maximum Product Subarray | 同时维护最大和最小乘积 |
| Best Time to Buy and Sell Stock | 可转成最大差值 |
| Maximum Subarray Circular | 环形最大子数组 |

### 常见坑点

- 全负数组不能把答案初始化成 0。
- 子数组必须连续。

## 高频模板 4：解码方法

### 秒判关键词

每个位置可由前一位或前两位转移。

### 母题

LC91 Decode Ways

### 标准代码

```java
public int numDecodings(String s) {
    int n = s.length();
    int[] dp = new int[n + 1];

    dp[0] = 1;
    dp[1] = s.charAt(0) == '0' ? 0 : 1;

    for (int i = 2; i <= n; i++) {
        char one = s.charAt(i - 1);
        if (one != '0') {
            dp[i] += dp[i - 1];
        }

        int two = Integer.parseInt(s.substring(i - 2, i));
        if (two >= 10 && two <= 26) {
            dp[i] += dp[i - 2];
        }
    }

    return dp[n];
}
```

### 一句话记忆

当前位置可由一位合法编码或两位合法编码转移。

### 常见坑点

- `'0'` 不能单独解码。
- `10` 和 `20` 合法，但 `30` 不合法。

---

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 状态没定义就写转移 | 先说 `dp[i]` 含义 |
| 初始化错 | 单独处理 base case |
| 全负最大子数组 | 答案初始化为 `nums[0]` |
| 环形打家劫舍 | 拆两段 |
| 过早空间优化 | 先写数组版 |

---

## 面试速记

- DP 先说状态：`dp[i]` 是什么。
- 再说选择：从哪些旧状态来。
- 最后说初始化。
- 线性 DP 常可滚动优化。
- 想不清空间优化时先写数组。
