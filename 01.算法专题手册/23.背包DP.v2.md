# 背包 DP

## 本章目标

训练“选或不选”的第一反应：看到容量、组合、凑和、最值，优先想到背包 DP。

---

# 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 每个元素只能用一次 | 0/1 背包 |
| 元素无限使用 | 完全背包 |
| 求方案数 | 组合 DP |
| 求最值 | max/min DP |
| 能否凑出 | boolean DP |

# 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 顺序写反 | 组合变排列 |
| 完全背包倒序 | 少算 |
| 初始化错误 | dp[0] 通常是 1 |

---

# 面试速记

- 先判断能否重复使用。
- 再判断求最值还是方案数。
- 遍历顺序决定题型。
