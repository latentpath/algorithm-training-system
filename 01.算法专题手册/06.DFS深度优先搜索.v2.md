# DFS 深度优先搜索

## 本章目标

训练“走到底再回退”的第一反应：树、图、网格、连通块、路径存在性，快速想到 DFS。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 树的高度 / 节点数 / 路径 | 树 DFS |
| 图是否连通 | DFS + visited |
| 网格岛屿数量 | 网格 DFS |
| 从一个点走到底 | DFS |
| 枚举所有方案 | 回溯，不是普通 DFS |
| 最短步数 | BFS 优先 |

## 高频模板 1：二叉树 DFS

### 秒判关键词

树、高度、路径、左右子树。

### 母题

LC104 Maximum Depth of Binary Tree

### 标准代码

```java
public int maxDepth(TreeNode root) {
    if (root == null) return 0;
    int left = maxDepth(root.left);
    int right = maxDepth(root.right);
    return Math.max(left, right) + 1;
}
```

### 一句话记忆

空节点给基础值，当前节点整合左右子树答案。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Same Tree | 同时递归两棵树 |
| Invert Binary Tree | 交换左右子树 |
| Path Sum | 带着剩余目标向下走 |

### 常见坑点

- 递归必须有空节点出口。
- 求子树信息通常用后序位置合并。

## 高频模板 2：图 DFS + visited

### 秒判关键词

连通分量、访问所有可达点、无向图。

### 母题

LC547 Number of Provinces

### 标准代码

```java
public int findCircleNum(int[][] isConnected) {
    int n = isConnected.length;
    boolean[] visited = new boolean[n];
    int ans = 0;

    for (int i = 0; i < n; i++) {
        if (!visited[i]) {
            dfs(isConnected, visited, i);
            ans++;
        }
    }
    return ans;
}

private void dfs(int[][] graph, boolean[] visited, int u) {
    visited[u] = true;
    for (int v = 0; v < graph.length; v++) {
        if (graph[u][v] == 1 && !visited[v]) {
            dfs(graph, visited, v);
        }
    }
}
```

### 一句话记忆

每次从未访问点出发，染完一个连通块。

### 高频变体

- Clone Graph：visited 改成 `Map<Node, Node>`。
- Course Schedule：有向图判环用三色标记。
- Number of Connected Components：邻接表更常见。

### 常见坑点

- 进入 DFS 时立刻标记 visited。
- 无向图可能从父节点走回来，必须防重复。

## 高频模板 3：网格 DFS

### 秒判关键词

岛屿、连通块、上下左右、染色。

### 母题

LC200 Number of Islands

### 标准代码

```java
public int numIslands(char[][] grid) {
    int m = grid.length, n = grid[0].length;
    int ans = 0;
    for (int r = 0; r < m; r++) {
        for (int c = 0; c < n; c++) {
            if (grid[r][c] == '1') {
                dfs(grid, r, c);
                ans++;
            }
        }
    }
    return ans;
}

private void dfs(char[][] grid, int r, int c) {
    if (r < 0 || r >= grid.length || c < 0 || c >= grid[0].length) return;
    if (grid[r][c] != '1') return;

    grid[r][c] = '0';
    dfs(grid, r + 1, c);
    dfs(grid, r - 1, c);
    dfs(grid, r, c + 1);
    dfs(grid, r, c - 1);
}
```

### 一句话记忆

遇到陆地就淹掉整座岛。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Max Area of Island | DFS 返回面积 |
| Flood Fill | 把同色区域改成新颜色 |
| Surrounded Regions | 从边界出发保护区域 |

### 常见坑点

- 先判越界，再访问数组。
- 染色要立刻做，否则会来回递归。
- 如果不能修改原数组，用 `visited`。

## 高频模板 4：路径存在性

### 秒判关键词

能否到达、是否存在路径、从起点走到终点。

### 母题

网格中是否存在从起点到终点的路径。

### 标准代码

```java
private boolean canReach(int[][] grid, boolean[][] visited, int r, int c) {
    if (r < 0 || r >= grid.length || c < 0 || c >= grid[0].length) return false;
    if (grid[r][c] == 1 || visited[r][c]) return false;
    if (r == grid.length - 1 && c == grid[0].length - 1) return true;

    visited[r][c] = true;
    return canReach(grid, visited, r + 1, c)
        || canReach(grid, visited, r - 1, c)
        || canReach(grid, visited, r, c + 1)
        || canReach(grid, visited, r, c - 1);
}
```

### 一句话记忆

存在性问题，一个方向成功就立刻返回。

### 高频变体

- Word Search：需要回溯撤销 visited。
- Maze：如果要最短距离，改 BFS。

### 常见坑点

- 普通连通性 visited 不撤销。
- 枚举所有路径才需要回溯撤销。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| DFS 无限递归 | 进入时标记 visited |
| 网格先访问再判越界 | 先判边界 |
| 最短路径用 DFS | 改 BFS |
| 回溯和 DFS 混淆 | 枚举方案才撤销选择 |
| 树递归无 base case | `root == null` |

## 一题多变体

```text
Maximum Depth
↓
Path Sum
↓
Diameter of Binary Tree
```

```text
Number of Islands
↓
Max Area of Island
↓
Surrounded Regions
```

## 面试速记

- DFS：一路走到底。
- 树 DFS：空节点出口，左右子树合并。
- 图 DFS：visited 防重复。
- 网格 DFS：越界 + 非目标格直接 return。
- 最短步数：优先 BFS。
