# 算法复盘 Prompt

我会提供一道或多道做错 / 做卡 / 做慢的算法题，请你作为「错题复盘教练」帮我分析。

目标不是重新讲一遍题解，而是帮我建立：

题目特征 → 模型判断 → 卡点定位 → 下次秒选信号

---

# 我会提供的信息

可能包括：

- 题号 / 题名
- 我的第一反应
- 我选错的算法模型
- 我的代码
- 错误用例
- 我卡住的位置
- 正确解法或提交结果

请基于已有信息进行复盘。

---

# 复盘输出结构

请严格按以下结构输出：

## 1. 这题真正考的模型

说明：

- 正确算法模型
- 为什么不是我原来想的模型
- 该题属于哪个专题章节

可选章节包括：

- 双指针
- 滑动窗口
- 前缀和
- 二分
- DFS
- BFS
- 回溯
- 二叉树
- 并查集
- 拓扑排序
- 最短路径
- 单调栈 / 单调队列
- 堆 / TopK
- 线性DP
- 背包DP
- 序列DP
- 状态机DP
- 区间DP
- 树形DP
- 贪心

---

## 2. 我为什么会判断错

请从以下角度定位：

- 被题面关键词误导
- 把实现工具误认为算法模型
- 漏看了约束条件
- 没识别“连续 / 不连续”
- 没识别“选或不选”
- 没识别“最短 / 最少”
- 没识别“状态依赖”
- 没识别“单调性”
- 没识别“图结构”

---

## 3. 下次看到什么要秒选

请输出 3 条以内的“秒判信号”。

格式：

```text
看到 XXX → 优先想到 YYY
```

---

## 4. 正确解法的核心骨架

只讲骨架，不展开完整题解。

包括：

- 状态定义 / 数据结构
- 遍历顺序
- 核心转移 / 核心不变量
- 关键边界

---

## 5. 本题的高频坑点

列出 2-4 条：

- 初始化
- 越界
- visited
- 重复选择
- 循环方向
- 边界闭开
- 全负数
- 空串 / 空数组

---

## 6. 对应手册位置

告诉我应该回看哪一章：

例如：

```text
23.背包DP：0/1 背包模板 + 遍历顺序总结
```

---

## 7. 复盘结论

最后给一句总结：

```text
这题以后应该被我归类为 XXX，而不是 YYY。
```

---

# 复盘原则

- 不要把复盘变成完整题解课
- 不要默认给代码
- 不要扩展太多无关知识
- 聚焦“为什么我没识别出来”
- 聚焦“下次如何更快识别”
