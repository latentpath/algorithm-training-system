# 单调栈与单调队列

## 本章目标

训练“淘汰无用候选”的第一反应：看到下一个更大、更小、窗口极值，快速想到单调结构。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 下一个更大元素 | 单调栈 |
| 右边第一个更小 | 单调栈 |
| 每个位置等多久 | 单调栈 |
| 柱状图最大矩形 | 单调递增栈 |
| 滑动窗口最大值 | 单调队列 |
| 固定窗口最小值 | 单调队列 |

## 高频模板 1：下一个更大元素

### 秒判关键词

next greater、右边第一个更大、等待多少天。

### 母题

LC739 Daily Temperatures

### 标准代码

```java
public int[] dailyTemperatures(int[] temperatures) {
    int n = temperatures.length;
    int[] ans = new int[n];
    Deque<Integer> stack = new ArrayDeque<>();

    for (int i = 0; i < n; i++) {
        while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) {
            int j = stack.pop();
            ans[j] = i - j;
        }
        stack.push(i);
    }
    return ans;
}
```

### 一句话记忆

栈里放还没找到更大值的下标。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Next Greater Element I | 用 Map 保存值到答案 |
| Next Greater Element II | 遍历两轮，处理循环数组 |
| Stock Span | 找左边连续小于等于自己的数量 |

### 常见坑点

- 栈存下标，答案通常需要距离。
- 当前元素可能解决多个旧元素，所以用 `while`。

## 高频模板 2：下一个更小 / 柱状图

### 秒判关键词

矩形面积、左右边界、第一个更小。

### 母题

LC84 Largest Rectangle in Histogram

### 标准代码

```java
public int largestRectangleArea(int[] heights) {
    int n = heights.length;
    Deque<Integer> stack = new ArrayDeque<>();
    int ans = 0;

    for (int i = 0; i <= n; i++) {
        int cur = i == n ? 0 : heights[i];
        while (!stack.isEmpty() && cur < heights[stack.peek()]) {
            int h = heights[stack.pop()];
            int left = stack.isEmpty() ? -1 : stack.peek();
            int width = i - left - 1;
            ans = Math.max(ans, h * width);
        }
        stack.push(i);
    }
    return ans;
}
```

### 一句话记忆

弹出一个柱子时，当前 i 是右边第一个更小，栈顶是左边第一个更小。

### 高频变体

- Maximal Rectangle：每一行转成柱状图。
- Sum of Subarray Minimums：计算每个元素作为最小值的贡献。

### 常见坑点

- 末尾加一个 0 清空栈。
- 宽度是 `i - left - 1`。

## 高频模板 3：循环数组单调栈

### 秒判关键词

循环数组、找下一个更大。

### 母题

LC503 Next Greater Element II

### 标准代码

```java
public int[] nextGreaterElements(int[] nums) {
    int n = nums.length;
    int[] ans = new int[n];
    Arrays.fill(ans, -1);
    Deque<Integer> stack = new ArrayDeque<>();

    for (int i = 0; i < 2 * n; i++) {
        int idx = i % n;
        while (!stack.isEmpty() && nums[idx] > nums[stack.peek()]) {
            ans[stack.pop()] = nums[idx];
        }
        if (i < n) stack.push(idx);
    }
    return ans;
}
```

### 一句话记忆

遍历两圈找答案，但只在第一圈入栈。

### 高频变体

- 环形温度 / 环形更大值：同样 `2n` + 取模。

### 常见坑点

- 第二圈不要再入栈，否则重复处理。
- 默认答案先填 `-1`。

## 高频模板 4：滑动窗口最大值

### 秒判关键词

窗口最大值、固定 k、区间极值。

### 母题

LC239 Sliding Window Maximum

### 标准代码

```java
public int[] maxSlidingWindow(int[] nums, int k) {
    int n = nums.length;
    int[] ans = new int[n - k + 1];
    Deque<Integer> deque = new ArrayDeque<>();

    for (int i = 0; i < n; i++) {
        while (!deque.isEmpty() && deque.peekFirst() <= i - k) {
            deque.pollFirst();
        }
        while (!deque.isEmpty() && nums[deque.peekLast()] <= nums[i]) {
            deque.pollLast();
        }
        deque.offerLast(i);
        if (i >= k - 1) {
            ans[i - k + 1] = nums[deque.peekFirst()];
        }
    }
    return ans;
}
```

### 一句话记忆

队首是最大值，队尾踢掉比当前更弱的候选。

### 高频变体

- 窗口最小值：比较方向反过来。
- Longest Continuous Subarray With Absolute Diff：一个最大队列 + 一个最小队列。

### 常见坑点

- 队列里存下标，才能判断过期。
- 过期从队首弹，弱候选从队尾弹。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 单调栈用 `if` | 用 `while` 弹完 |
| 栈里存值 | 多数题存下标 |
| 柱状图不清栈 | 末尾哨兵 0 |
| 循环数组重复入栈 | 只第一圈入栈 |
| 单调队列不会过期 | 用下标判断 `<= i-k` |

## 面试速记

- 单调栈：解决“某个元素左右第一个更大 / 更小”。
- 单调队列：解决“滑动窗口最大 / 最小”。
- 栈弹出时结算答案。
- 队首提供窗口答案。
- 存下标比存值更稳。
