# Flood Fill 网格染色

## 本章目标

训练网格连通块反应：看到岛屿、染色、区域、边界包围，立刻想到 DFS/BFS 把一片区域处理掉。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 岛屿数量 | 遇到陆地就染掉 |
| 最大岛屿面积 | DFS 返回面积 |
| 同色区域换色 | Flood Fill |
| 被包围区域 | 从边界反向染色 |
| 最近距离 / 最短扩散 | BFS |
| 多个源头同时扩散 | 多源 BFS |

## 高频模板 1：标准染色 DFS

### 秒判关键词

连通块、岛屿、上下左右、访问后改值。

### 母题

LC200 Number of Islands

### 标准代码

```java
public int numIslands(char[][] grid) {
    int m = grid.length, n = grid[0].length;
    int ans = 0;
    for (int r = 0; r < m; r++) {
        for (int c = 0; c < n; c++) {
            if (grid[r][c] == '1') {
                fill(grid, r, c);
                ans++;
            }
        }
    }
    return ans;
}

private void fill(char[][] grid, int r, int c) {
    if (r < 0 || r >= grid.length || c < 0 || c >= grid[0].length) return;
    if (grid[r][c] != '1') return;

    grid[r][c] = '0';
    fill(grid, r + 1, c);
    fill(grid, r - 1, c);
    fill(grid, r, c + 1);
    fill(grid, r, c - 1);
}
```

### 一句话记忆

遇到一个格子，就把它所在的一整片染掉。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Max Area of Island | DFS 返回面积 |
| Flood Fill | 只染原颜色相同的区域 |
| Number of Closed Islands | 碰边界就不是封闭岛 |

### 常见坑点

- 染色必须在递归前做。
- 先判断越界，再访问 `grid[r][c]`。
- 如果新颜色等于旧颜色，Flood Fill 要提前返回。

## 高频模板 2：面积 / 计数型 DFS

### 秒判关键词

最大面积、连通块大小、区域数量。

### 母题

LC695 Max Area of Island

### 标准代码

```java
public int maxAreaOfIsland(int[][] grid) {
    int ans = 0;
    for (int r = 0; r < grid.length; r++) {
        for (int c = 0; c < grid[0].length; c++) {
            if (grid[r][c] == 1) {
                ans = Math.max(ans, area(grid, r, c));
            }
        }
    }
    return ans;
}

private int area(int[][] grid, int r, int c) {
    if (r < 0 || r >= grid.length || c < 0 || c >= grid[0].length) return 0;
    if (grid[r][c] != 1) return 0;

    grid[r][c] = 0;
    return 1
        + area(grid, r + 1, c)
        + area(grid, r - 1, c)
        + area(grid, r, c + 1)
        + area(grid, r, c - 1);
}
```

### 一句话记忆

染掉一个格子，同时把它贡献的面积返回。

### 高频变体

- 岛屿周长：看每个陆地的四条边。
- 最大人工岛：先给岛编号，再尝试翻转 0。

### 常见坑点

- 面积题不要重复访问同一格。
- 如果不能修改原图，用 `visited`。

## 高频模板 3：边界反向染色

### 秒判关键词

被包围、不能从边界到达、封闭区域。

### 母题

LC130 Surrounded Regions

### 标准代码

```java
public void solve(char[][] board) {
    int m = board.length, n = board[0].length;
    for (int r = 0; r < m; r++) {
        mark(board, r, 0);
        mark(board, r, n - 1);
    }
    for (int c = 0; c < n; c++) {
        mark(board, 0, c);
        mark(board, m - 1, c);
    }

    for (int r = 0; r < m; r++) {
        for (int c = 0; c < n; c++) {
            if (board[r][c] == 'O') board[r][c] = 'X';
            if (board[r][c] == '#') board[r][c] = 'O';
        }
    }
}

private void mark(char[][] board, int r, int c) {
    if (r < 0 || r >= board.length || c < 0 || c >= board[0].length) return;
    if (board[r][c] != 'O') return;
    board[r][c] = '#';
    mark(board, r + 1, c);
    mark(board, r - 1, c);
    mark(board, r, c + 1);
    mark(board, r, c - 1);
}
```

### 一句话记忆

先保护边界能到的区域，剩下的才是被包围的。

### 高频变体

- Number of Enclaves：从边界陆地出发淹掉，剩下计数。
- Closed Islands：边界连通的不是封闭岛。

### 常见坑点

- 不要从内部判断是否被包围，边界反向更稳。
- 临时标记要最后恢复。

## 高频模板 4：网格 BFS 染色

### 秒判关键词

网格扩散、最短距离、多源。

### 母题

LC994 Rotting Oranges

### 标准代码

```java
public int orangesRotting(int[][] grid) {
    Queue<int[]> queue = new ArrayDeque<>();
    int fresh = 0;
    for (int r = 0; r < grid.length; r++) {
        for (int c = 0; c < grid[0].length; c++) {
            if (grid[r][c] == 2) queue.offer(new int[]{r, c});
            if (grid[r][c] == 1) fresh++;
        }
    }

    int[][] dirs = {{1,0},{-1,0},{0,1},{0,-1}};
    int minute = 0;
    while (!queue.isEmpty() && fresh > 0) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int[] cur = queue.poll();
            for (int[] d : dirs) {
                int nr = cur[0] + d[0], nc = cur[1] + d[1];
                if (nr < 0 || nr >= grid.length || nc < 0 || nc >= grid[0].length) continue;
                if (grid[nr][nc] != 1) continue;
                grid[nr][nc] = 2;
                fresh--;
                queue.offer(new int[]{nr, nc});
            }
        }
        minute++;
    }
    return fresh == 0 ? minute : -1;
}
```

### 一句话记忆

要最短扩散时间，用 BFS 按层染。

### 高频变体

- 01 Matrix：所有 0 入队。
- Walls and Gates：所有门入队。

### 常见坑点

- 多源同时入队，不是一个个源点跑。
- 入队时修改状态，避免重复入队。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 递归爆栈风险 | 大网格可改 BFS |
| 新旧颜色相同 | Flood Fill 提前返回 |
| 忘记染色 | 无限递归 |
| 被包围区域从内部做 | 从边界反向染 |
| 最短距离用 DFS | 改 BFS |

## 一题多变体

```text
Number of Islands
↓
Max Area of Island
↓
Closed Islands
↓
Surrounded Regions
```

```text
Flood Fill
↓
Rotting Oranges
↓
01 Matrix
```

## 面试速记

- 连通块：DFS 染掉。
- 面积：DFS 返回计数。
- 包围：边界反向保护。
- 最短扩散：BFS。
- 网格四件套：越界、目标值、染色、四方向。
