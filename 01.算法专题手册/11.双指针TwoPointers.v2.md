# 双指针 Two Pointers

## 本章目标

训练“两个位置一起移动”的第一反应：看到有序、回文、删除保序、原地修改，快速想到双指针。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 有序数组两数和 | 左右指针 |
| 回文判断 | 左右指针 |
| 反转数组 / 字符串 | 左右指针 |
| 删除元素 + 保序 | 快慢指针 |
| 原地去重 | 快慢指针 |
| 链表倒数第 k 个 | 快慢指针 |
| 判断链表环 | Floyd 快慢指针 |

## 高频模板 1：左右夹逼

### 秒判关键词

有序数组、两数和、回文、首尾向中间。

### 母题

LC167 Two Sum II

### 标准代码

```java
public int[] twoSum(int[] numbers, int target) {
    int left = 0, right = numbers.length - 1;
    while (left < right) {
        int sum = numbers[left] + numbers[right];
        if (sum == target) {
            return new int[]{left + 1, right + 1};
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    return new int[0];
}
```

### 一句话记忆

和小了左指针右移，和大了右指针左移。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Valid Palindrome | 不求和，只比较字符 |
| Reverse String | 比较变交换 |
| 3Sum | 排序后固定一个数，再左右夹逼 |

### 常见坑点

- 左右夹逼通常要求有序或可排序。
- 3Sum 要跳过重复固定值和重复左右值。

## 高频模板 2：快慢指针原地覆盖

### 秒判关键词

删除、去重、移动零、保序、返回新长度。

### 母题

LC27 Remove Element

### 标准代码

```java
public int removeElement(int[] nums, int val) {
    int slow = 0;
    for (int fast = 0; fast < nums.length; fast++) {
        if (nums[fast] != val) {
            nums[slow] = nums[fast];
            slow++;
        }
    }
    return slow;
}
```

### 一句话记忆

`fast` 找有效元素，`slow` 维护写入位置。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Move Zeroes | 先写非零，最后补零 |
| Remove Duplicates | 与 `nums[slow - 1]` 比 |
| Remove Duplicates II | 与 `nums[slow - 2]` 比 |

### 常见坑点

- `slow` 代表下一个写入位置。
- 只在写入有效元素后 `slow++`。

## 高频模板 3：链表快慢指针

### 秒判关键词

倒数第 k 个、中点、环。

### 母题

LC19 Remove Nth Node From End of List

### 标准代码

```java
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    ListNode fast = dummy, slow = dummy;

    for (int i = 0; i < n; i++) {
        fast = fast.next;
    }
    while (fast.next != null) {
        fast = fast.next;
        slow = slow.next;
    }
    slow.next = slow.next.next;
    return dummy.next;
}
```

### 一句话记忆

快指针先走 n 步，慢指针停在待删节点前一个。

### 高频变体

- Middle of Linked List：快走两步，慢走一步。
- Linked List Cycle：快慢相遇说明有环。
- Palindrome Linked List：找中点 + 反转后半段。

### 常见坑点

- 删除头节点时必须用 dummy。
- 删除倒数第 n 个时，慢指针要停在前驱。

## 高频模板 4：Floyd 判环

### 秒判关键词

链表是否有环、找环入口、快慢相遇。

### 母题

LC141 Linked List Cycle

### 标准代码

```java
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) return true;
    }
    return false;
}
```

### 一句话记忆

快的每次走两步，能追上慢的就有环。

### 高频变体

- Linked List Cycle II：相遇后一个指针回 head，同速走到入口。
- Find Duplicate Number：数组下标当链表指针。

### 常见坑点

- while 条件必须检查 `fast` 和 `fast.next`。
- 链表节点比较用引用，不是值。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 无序数组直接左右夹逼 | 先排序或换哈希 |
| 原地删除移动错指针 | fast 扫描，slow 写 |
| 链表删除头节点 | dummy |
| 找倒数第 n 个 off-by-one | 慢指针停前驱 |
| 判环空指针异常 | 检查 `fast != null && fast.next != null` |

## 面试速记

- 有序 + 两端：左右夹逼。
- 删除保序：快慢覆盖。
- 链表倒数：快先走。
- 链表环：Floyd。
- 头节点可能变：dummy。
