# 回溯 Backtracking

## 本章目标

训练“枚举所有方案”的第一反应：看到子集、组合、排列、分割，快速套“路径 + 选择 + 结束条件”。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 所有子集 | 选 / 不选 或 start 枚举 |
| 所有组合 | `start` 防重复选 |
| 所有排列 | `used[]` 防重复用 |
| 可重复选择 | 递归仍从 `i` 开始 |
| 不可重复选择 | 递归从 `i + 1` 开始 |
| 输入有重复元素 | 排序 + 同层去重 |
| 分割字符串 | 枚举切割点 |

## 高频模板 1：子集 / 组合型

### 秒判关键词

所有子集、所有组合、不关心顺序。

### 母题

LC78 Subsets

### 标准代码

```java
public List<List<Integer>> subsets(int[] nums) {
    List<List<Integer>> ans = new ArrayList<>();
    backtrack(nums, 0, new ArrayList<>(), ans);
    return ans;
}

private void backtrack(int[] nums, int start, List<Integer> path, List<List<Integer>> ans) {
    ans.add(new ArrayList<>(path));

    for (int i = start; i < nums.length; i++) {
        path.add(nums[i]);
        backtrack(nums, i + 1, path, ans);
        path.remove(path.size() - 1);
    }
}
```

### 一句话记忆

组合不看顺序，用 `start` 保证只往后选。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Combinations | path 长度到 k 才加入 |
| Combination Sum III | 加上 sum 限制 |
| Subsets II | 排序 + 同层去重 |

### 常见坑点

- 加入答案时要 `new ArrayList<>(path)`。
- `start` 控制不回头选。

## 高频模板 2：排列型

### 秒判关键词

所有排列、顺序不同算不同。

### 母题

LC46 Permutations

### 标准代码

```java
public List<List<Integer>> permute(int[] nums) {
    List<List<Integer>> ans = new ArrayList<>();
    boolean[] used = new boolean[nums.length];
    backtrack(nums, used, new ArrayList<>(), ans);
    return ans;
}

private void backtrack(int[] nums, boolean[] used, List<Integer> path, List<List<Integer>> ans) {
    if (path.size() == nums.length) {
        ans.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        used[i] = true;
        path.add(nums[i]);
        backtrack(nums, used, path, ans);
        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

### 一句话记忆

排列每一层都从头看，用 `used` 防止同一元素重复使用。

### 高频变体

- Permutations II：排序 + 同层跳过重复元素。
- Letter Case Permutation：每个位置有多个选择。

### 常见坑点

- 撤销顺序要完整：删 path，恢复 used。
- 有重复元素时只用 `used` 不够，还要去重。

## 高频模板 3：组合总和

### 秒判关键词

数字可重复使用、凑目标和。

### 母题

LC39 Combination Sum

### 标准代码

```java
public List<List<Integer>> combinationSum(int[] candidates, int target) {
    List<List<Integer>> ans = new ArrayList<>();
    backtrack(candidates, target, 0, new ArrayList<>(), ans);
    return ans;
}

private void backtrack(int[] nums, int remain, int start, List<Integer> path, List<List<Integer>> ans) {
    if (remain == 0) {
        ans.add(new ArrayList<>(path));
        return;
    }
    if (remain < 0) return;

    for (int i = start; i < nums.length; i++) {
        path.add(nums[i]);
        backtrack(nums, remain - nums[i], i, path, ans);
        path.remove(path.size() - 1);
    }
}
```

### 一句话记忆

可重复选，下一层还是从 `i` 开始。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Combination Sum II | 每个数只能用一次，下一层 `i + 1` |
| Combination Sum III | 固定数量 k |
| Target Sum | 可转 DP，也可回溯入门 |

### 常见坑点

- 可重复和不可重复只差下一层起点。
- 排序后如果 `nums[i] > remain` 可以剪枝。

## 高频模板 4：分割型回溯

### 秒判关键词

分割字符串、所有切法、每段满足条件。

### 母题

LC131 Palindrome Partitioning

### 标准代码

```java
public List<List<String>> partition(String s) {
    List<List<String>> ans = new ArrayList<>();
    backtrack(s, 0, new ArrayList<>(), ans);
    return ans;
}

private void backtrack(String s, int start, List<String> path, List<List<String>> ans) {
    if (start == s.length()) {
        ans.add(new ArrayList<>(path));
        return;
    }

    for (int end = start; end < s.length(); end++) {
        if (!isPalindrome(s, start, end)) continue;
        path.add(s.substring(start, end + 1));
        backtrack(s, end + 1, path, ans);
        path.remove(path.size() - 1);
    }
}
```

### 一句话记忆

分割题就是枚举当前这一刀切到哪里。

### 高频变体

- Restore IP Addresses：每段长度 1 到 3 且合法。
- Word Break II：每段必须在字典里。

### 常见坑点

- 结束条件是 `start == s.length()`。
- `substring` 右边界不包含。

## 高频模板 5：同层去重

### 秒判关键词

输入有重复元素、答案不能重复。

### 母题

LC90 Subsets II

### 标准代码片段

```java
Arrays.sort(nums);
for (int i = start; i < nums.length; i++) {
    if (i > start && nums[i] == nums[i - 1]) continue;
    path.add(nums[i]);
    backtrack(nums, i + 1, path, ans);
    path.remove(path.size() - 1);
}
```

### 一句话记忆

同一层相同选择只用第一次。

### 高频变体

- Subsets II：同层去重。
- Combination Sum II：同层去重 + 不可重复选。
- Permutations II：排序 + `used[i - 1]` 控制重复。

### 常见坑点

- 去重前必须排序。
- 去重是“同层去重”，不是所有相同值都不能用。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 答案全变成空 | 加入时复制 path |
| 忘记撤销选择 | `add` 后必须 `remove` |
| 组合写成排列 | 用 `start` |
| 排列漏元素 | 用 `used[]` |
| 重复答案 | 排序 + 同层去重 |

## 一题多变体

## 面试速记

- 回溯三件套：路径、选择、结束条件。
- 组合：`start`。
- 排列：`used`。
- 可重复：下一层 `i`。
- 不可重复：下一层 `i + 1`。
- 有重复：排序 + 同层去重。
