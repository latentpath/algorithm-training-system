# 动态规划基础：线性 DP

## 本章目标

训练一维 DP 的第一反应：看到“到当前位置的最优值 / 方案数”，优先想到线性 DP。

---

# 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 到第 i 个位置的最优值 | 一维 DP |
| 依赖前几个位置 | 线性转移 |
| 不能选相邻 | 打家劫舍 |
| 连续子数组最优值 | Kadane |
| 爬楼梯 | Fibonacci 型 DP |

# 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 状态定义不清 | 先定义 dp[i] |
| 初始化混乱 | 单独处理 base case |
| 空间优化过早 | 先写完整 DP |

---

# 面试速记

- 线性 DP 通常从左往右。
- 当前位置只依赖前面少量状态。
- 先定义状态，再写转移。
