# 数组 + 字符串 + 链表

## 本章目标

训练最基础的线性结构反应：删除、反转、合并、链表指针变化时，先想到稳定模板。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 删除元素 + 保持顺序 | 快慢指针 |
| 原地覆盖数组 | `fast` 扫描，`slow` 写入 |
| 反转数组 / 字符串 | 左右指针 |
| 两个有序数组 / 链表 | 双指针合并 |
| 链表头节点可能变化 | dummy node |
| 链表反转 | 先存 `next`，再改指针 |
| 字符串循环拼接 | `StringBuilder` |

## 高频模板 1：原地删除 / 覆盖

### 秒判关键词

删除、去重、保序、原地、返回新长度。

### 母题

LC27 Remove Element

### 标准代码

```java
public int removeElement(int[] nums, int val) {
    int slow = 0;
    for (int fast = 0; fast < nums.length; fast++) {
        if (nums[fast] != val) {
            nums[slow] = nums[fast];
            slow++;
        }
    }
    return slow;
}
```

### 一句话记忆

`fast` 扫描全数组，`slow` 只写有效区。

### 高频变体

| 题目 | 变化点 |
|---|---|
| LC283 Move Zeroes | 非零写前面，最后补零 |
| LC26 Remove Duplicates | `nums[fast] != nums[slow - 1]` 才写 |
| LC80 Remove Duplicates II | 允许两个：`nums[fast] != nums[slow - 2]` |

### 常见坑点

- `slow++` 只在写入后发生。
- 返回的是新长度，不是数组本身。
- 覆盖区只保证 `[0, slow)` 有效。

## 高频模板 2：左右指针反转

### 秒判关键词

反转、回文、首尾交换、夹逼。

### 母题

LC344 Reverse String

### 标准代码

```java
public void reverseString(char[] s) {
    int left = 0, right = s.length - 1;
    while (left < right) {
        char tmp = s[left];
        s[left] = s[right];
        s[right] = tmp;
        left++;
        right--;
    }
}
```

### 一句话记忆

左右夹逼，交换后同时收缩。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Valid Palindrome | 不交换，只比较 |
| Reverse Words | 先整体处理，再按词处理 |
| Two Sum II | 根据和大小移动左右指针 |

### 常见坑点

- 条件是 `left < right`。
- 字符串不可变时不要循环 `+` 拼接。
- 回文题常要跳过非字母数字字符。

## 高频模板 3：链表 dummy node

### 秒判关键词

删除链表节点、合并链表、头节点可能被删。

### 母题

LC203 Remove Linked List Elements

### 标准代码

```java
public ListNode removeElements(ListNode head, int val) {
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    ListNode cur = dummy;

    while (cur.next != null) {
        if (cur.next.val == val) {
            cur.next = cur.next.next;
        } else {
            cur = cur.next;
        }
    }
    return dummy.next;
}
```

### 一句话记忆

头节点可能变化，先挂 dummy。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Merge Two Sorted Lists | dummy 后面接较小节点 |
| Remove Nth From End | dummy + 快慢指针 |
| Partition List | 两个 dummy 分链再拼接 |

### 常见坑点

- 删除节点后不要立刻 `cur = cur.next`，因为新 `cur.next` 还没检查。
- 返回 `dummy.next`，不是 `head`。

## 高频模板 4：链表反转

### 秒判关键词

反转链表、局部反转、K 个一组。

### 母题

LC206 Reverse Linked List

### 标准代码

```java
public ListNode reverseList(ListNode head) {
    ListNode prev = null;
    ListNode cur = head;
    while (cur != null) {
        ListNode next = cur.next;
        cur.next = prev;
        prev = cur;
        cur = next;
    }
    return prev;
}
```

### 一句话记忆

先存 `next`，再反转 `cur.next`。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Reverse Linked List II | 只反转区间 |
| Swap Nodes in Pairs | 每次处理两个节点 |
| Reverse Nodes in k-Group | 先确认够 `k` 个 |

### 常见坑点

- 不先存 `next` 会断链。
- 最后新头是 `prev`。
- 局部反转要保存区间前驱和区间后继。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 删除数组元素时跳过检查 | 写入后再移动 `slow` |
| 链表头节点被删 | dummy |
| 链表反转丢后续节点 | 先存 `next` |
| 字符串循环拼接超时 | `StringBuilder` |
| 合并链表忘接尾巴 | 循环后接剩余链表 |

## 面试速记

- 数组删除：`fast` 找，`slow` 写。
- 反转 / 回文：左右夹逼。
- 链表头会变：dummy。
- 链表反转：`next -> reverse -> move`。
- Java 字符串拼接：循环中用 `StringBuilder`。
