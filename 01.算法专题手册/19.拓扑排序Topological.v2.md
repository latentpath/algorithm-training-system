# 拓扑排序 Topological Sort

## 本章目标

训练“依赖顺序”的第一反应：看到课程先修、任务依赖、有向图能否完成，快速想到拓扑排序。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 课程先修关系 | 拓扑排序 |
| 任务有依赖顺序 | 拓扑排序 |
| 有向图是否有环 | 拓扑 / DFS 三色 |
| 输出一个合法顺序 | Kahn BFS |
| 字典外星人顺序 | 建图 + 拓扑 |
| 无向图连通 | DFS / 并查集，不是拓扑 |

## 高频模板 1：Kahn 入度 BFS

### 秒判关键词

有向依赖、能否完成、输出顺序。

### 母题

LC207 Course Schedule

### 标准代码

```java
public boolean canFinish(int numCourses, int[][] prerequisites) {
    List<Integer>[] graph = new ArrayList[numCourses];
    for (int i = 0; i < numCourses; i++) graph[i] = new ArrayList<>();
    int[] indegree = new int[numCourses];

    for (int[] p : prerequisites) {
        int course = p[0], pre = p[1];
        graph[pre].add(course);
        indegree[course]++;
    }

    Queue<Integer> queue = new ArrayDeque<>();
    for (int i = 0; i < numCourses; i++) {
        if (indegree[i] == 0) queue.offer(i);
    }

    int count = 0;
    while (!queue.isEmpty()) {
        int cur = queue.poll();
        count++;
        for (int next : graph[cur]) {
            indegree[next]--;
            if (indegree[next] == 0) queue.offer(next);
        }
    }
    return count == numCourses;
}
```

### 一句话记忆

入度为 0 的点先做，做完就删掉它的影响。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Course Schedule II | 记录出队顺序 |
| Alien Dictionary | 字符建图后拓扑 |
| Minimum Height Trees | 无向图剥叶子，思路类似 BFS |

### 常见坑点

- 边方向：先修课 `pre -> course`。
- 出队数量少于节点数说明有环。

## 高频模板 2：输出拓扑序

### 秒判关键词

返回学习顺序、任务执行顺序、合法排列。

### 母题

LC210 Course Schedule II

### 标准代码

```java
public int[] findOrder(int numCourses, int[][] prerequisites) {
    List<Integer>[] graph = new ArrayList[numCourses];
    for (int i = 0; i < numCourses; i++) graph[i] = new ArrayList<>();
    int[] indegree = new int[numCourses];

    for (int[] p : prerequisites) {
        graph[p[1]].add(p[0]);
        indegree[p[0]]++;
    }

    Queue<Integer> queue = new ArrayDeque<>();
    for (int i = 0; i < numCourses; i++) {
        if (indegree[i] == 0) queue.offer(i);
    }

    int[] order = new int[numCourses];
    int idx = 0;
    while (!queue.isEmpty()) {
        int cur = queue.poll();
        order[idx++] = cur;
        for (int next : graph[cur]) {
            if (--indegree[next] == 0) queue.offer(next);
        }
    }
    return idx == numCourses ? order : new int[0];
}
```

### 一句话记忆

能出队的顺序，就是一个合法拓扑序。

### 高频变体

- Parallel Courses：层序 BFS 统计学期数。
- Sequence Reconstruction：检查拓扑序是否唯一。

### 常见坑点

- 多个入度 0 点时，普通拓扑序不唯一。
- 要求字典序最小，用优先队列替代普通队列。

## 高频模板 3：DFS 三色判环

### 秒判关键词

有向图成环、递归访问状态。

### 母题

Course Schedule 的 DFS 写法

### 标准代码

```java
public boolean canFinish(int numCourses, int[][] prerequisites) {
    List<Integer>[] graph = new ArrayList[numCourses];
    for (int i = 0; i < numCourses; i++) graph[i] = new ArrayList<>();
    for (int[] p : prerequisites) {
        graph[p[1]].add(p[0]);
    }

    int[] state = new int[numCourses]; // 0 未访问，1 访问中，2 已完成
    for (int i = 0; i < numCourses; i++) {
        if (hasCycle(graph, state, i)) return false;
    }
    return true;
}

private boolean hasCycle(List<Integer>[] graph, int[] state, int u) {
    if (state[u] == 1) return true;
    if (state[u] == 2) return false;

    state[u] = 1;
    for (int v : graph[u]) {
        if (hasCycle(graph, state, v)) return true;
    }
    state[u] = 2;
    return false;
}
```

### 一句话记忆

DFS 走回“访问中”的点，就发现有向环。

### 高频变体

- Find Eventual Safe States：反向图 BFS 或 DFS 三色。
- Alien Dictionary：也可 DFS 判环。

### 常见坑点

- 三色标记中，`1` 是当前递归栈里。
- 无向图判环不能直接照搬有向三色模板。

## 高频模板 4：外星词典建图

### 秒判关键词

根据单词顺序推字符顺序。

### 母题

LC269 Alien Dictionary

### 标准思路

```text
1. 收集所有字符。
2. 比较相邻两个单词，找到第一个不同字符 a、b。
3. 建边 a -> b。
4. 如果前一个单词更长且后一个是它前缀，直接非法。
5. 对字符图做拓扑排序。
```

### 一句话记忆

只用相邻单词的第一个不同字符建边。

### 高频变体

- Verifying an Alien Dictionary：已有顺序，只需比较。
- 不要求输出顺序时，只判断是否有环。

### 常见坑点

- 不要从一个单词内部相邻字符建边。
- 前缀非法情况必须单独处理。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 边方向写反 | 先修 `pre -> course` |
| 入度初始化漏节点 | 所有节点都要建空表 |
| 出队少了没判断 | `count == n` |
| 有向无向混淆 | 拓扑只用于有向依赖 |
| 外星词典乱建边 | 相邻单词第一个不同字符 |

## 面试速记

- 拓扑排序：有向依赖顺序。
- Kahn：入度 0 入队。
- 出队数不够：有环。
- 输出顺序：记录出队顺序。
- DFS 判环：遇到访问中节点。
