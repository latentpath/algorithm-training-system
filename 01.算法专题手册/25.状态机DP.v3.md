# 状态机 DP

## 本章目标

训练“状态切换”的第一反应：看到持有 / 未持有、买卖、冷冻期、多状态，优先想到状态机 DP。

---

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 股票买卖 | 状态机 DP |
| 持有 / 未持有 | 两状态 |
| 冷冻期 | 多状态 |
| 手续费 | 状态转移成本 |
| 最多 k 次交易 | 三维状态 |
| 多个互斥状态 | 状态机 |

---

## 高频模板 1：股票 I

### 秒判关键词

只允许一次买卖，最大利润。

### 母题

LC121 Best Time to Buy and Sell Stock

### 标准代码

```java
public int maxProfit(int[] prices) {
    int minPrice = Integer.MAX_VALUE;
    int ans = 0;

    for (int p : prices) {
        minPrice = Math.min(minPrice, p);
        ans = Math.max(ans, p - minPrice);
    }

    return ans;
}
```

### 一句话记忆

维护历史最低买入价。

---

## 高频模板 2：股票 II

### 秒判关键词

可以多次交易，不能同时持有多支股票。

### 母题

LC122 Best Time to Buy and Sell Stock II

### 标准代码

```java
public int maxProfit(int[] prices) {
    int hold = -prices[0];
    int cash = 0;

    for (int i = 1; i < prices.length; i++) {
        int newHold = Math.max(hold, cash - prices[i]);
        int newCash = Math.max(cash, hold + prices[i]);

        hold = newHold;
        cash = newCash;
    }

    return cash;
}
```

### 一句话记忆

每天只有两个状态：持有股票，或手里是现金。

### 常见坑点

- 更新时最好用临时变量，避免覆盖。
- 最终答案一定是 `cash`，不是 `hold`。

## 高频模板 3：含冷冻期

### 秒判关键词

卖出后下一天不能买入。

### 母题

LC309 Best Time to Buy and Sell Stock with Cooldown

### 标准代码

```java
public int maxProfit(int[] prices) {
    int hold = -prices[0];
    int sold = 0;
    int rest = 0;

    for (int i = 1; i < prices.length; i++) {
        int newHold = Math.max(hold, rest - prices[i]);
        int newSold = hold + prices[i];
        int newRest = Math.max(rest, sold);

        hold = newHold;
        sold = newSold;
        rest = newRest;
    }

    return Math.max(sold, rest);
}
```

### 一句话记忆

卖出后进入冷冻，冷冻后才能重新买。

## 高频模板 4：含手续费

### 秒判关键词

每次交易扣手续费。

### 母题

LC714 Best Time to Buy and Sell Stock with Transaction Fee

### 标准代码

```java
public int maxProfit(int[] prices, int fee) {
    int hold = -prices[0];
    int cash = 0;

    for (int i = 1; i < prices.length; i++) {
        int newHold = Math.max(hold, cash - prices[i]);
        int newCash = Math.max(cash, hold + prices[i] - fee);

        hold = newHold;
        cash = newCash;
    }

    return cash;
}
```

### 一句话记忆

手续费可以在买入扣，也可以在卖出扣，但只能扣一次。

## 高频模板 5：最多 k 次交易

### 秒判关键词

最多 k 次交易，交易次数受限。

### 母题

LC188 Best Time to Buy and Sell Stock IV

### 标准代码

```java
public int maxProfit(int k, int[] prices) {
    int n = prices.length;
    if (n == 0 || k == 0) return 0;

    if (k >= n / 2) {
        int profit = 0;
        for (int i = 1; i < n; i++) {
            if (prices[i] > prices[i - 1]) {
                profit += prices[i] - prices[i - 1];
            }
        }
        return profit;
    }

    int[][] buy = new int[k + 1][n];
    int[][] sell = new int[k + 1][n];

    for (int t = 1; t <= k; t++) {
        buy[t][0] = -prices[0];
    }

    for (int t = 1; t <= k; t++) {
        for (int i = 1; i < n; i++) {
            buy[t][i] = Math.max(buy[t][i - 1], sell[t - 1][i - 1] - prices[i]);
            sell[t][i] = Math.max(sell[t][i - 1], buy[t][i - 1] + prices[i]);
        }
    }

    return sell[k][n - 1];
}
```

### 一句话记忆

买入消耗一次交易额度，卖出完成收益。

---

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 状态含义混乱 | 先写 hold / cash / sold / rest |
| 更新顺序覆盖 | 用临时变量 |
| 交易次数理解错 | 买入或完成一次买卖时统一定义 |
| 最终返回 hold | 通常返回未持有状态 |
| 手续费重复扣 | 买入或卖出扣一次即可 |

---

## 面试速记

- 状态机 DP 本质是有限状态转移。
- 股票问题是最经典模板。
- 先定义状态，再写转移。
- 持有和未持有通常是核心状态。
