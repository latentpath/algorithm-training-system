# 树形 DP

## 本章目标

训练“子树汇总”的第一反应：看到树上路径、父子依赖、选或不选节点，优先想到树形 DP。

---

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 子树信息 | 树形 DP |
| 父子依赖 | DFS + DP |
| 树上选点 | 树形 DP |
| 树上路径最大值 | 树 DP |
| 子节点向父节点返回信息 | 后序 DFS |
| 重复 DFS 子问题 | 记忆化搜索 |

## 高频模板 1：树上打家劫舍

### 秒判关键词

树上选点、选父不能选子。

### 母题

LC337 House Robber III

### 标准代码

```java
public int rob(TreeNode root) {
    int[] ans = dfs(root);
    return Math.max(ans[0], ans[1]);
}

private int[] dfs(TreeNode node) {
    if (node == null) {
        return new int[]{0, 0};
    }

    int[] left = dfs(node.left);
    int[] right = dfs(node.right);

    int rob = node.val + left[0] + right[0];
    int skip = Math.max(left[0], left[1]) + Math.max(right[0], right[1]);

    return new int[]{skip, rob};
}
```

### 一句话记忆

偷当前，就不能偷孩子；不偷当前，孩子可偷可不偷。

## 高频模板 2：二叉树最大路径和

### 秒判关键词

路径可以从任意节点开始和结束，求最大路径和。

### 母题

LC124 Binary Tree Maximum Path Sum

### 标准代码

```java
private int ans;

public int maxPathSum(TreeNode root) {
    ans = Integer.MIN_VALUE;
    gain(root);
    return ans;
}

private int gain(TreeNode node) {
    if (node == null) {
        return 0;
    }

    int left = Math.max(0, gain(node.left));
    int right = Math.max(0, gain(node.right));

    ans = Math.max(ans, node.val + left + right);

    return node.val + Math.max(left, right);
}
```

### 一句话记忆

向上只能返回单边贡献，全局答案可以经过左右两边。

### 常见坑点

- 返回值不是全局路径，而是给父节点用的单边最大贡献。
- 负贡献要丢掉。

## 高频模板 3：树直径

### 秒判关键词

树中任意两点最长路径。

### 母题

LC543 Diameter of Binary Tree

### 标准代码

```java
private int diameter;

public int diameterOfBinaryTree(TreeNode root) {
    diameter = 0;
    depth(root);
    return diameter;
}

private int depth(TreeNode node) {
    if (node == null) {
        return 0;
    }

    int left = depth(node.left);
    int right = depth(node.right);

    diameter = Math.max(diameter, left + right);

    return Math.max(left, right) + 1;
}
```

### 一句话记忆

每个节点尝试作为路径拐点：左深度 + 右深度。

## 高频模板 4：最长递增路径

### 秒判关键词

网格中重复 DFS 子问题，求最长路径。

### 母题

LC329 Longest Increasing Path in a Matrix

### 标准代码

```java
private int[][] memo;
private int[][] dirs = {{1, 0}, {-1, 0}, {0, 1}, {0, -1}};

public int longestIncreasingPath(int[][] matrix) {
    int m = matrix.length;
    int n = matrix[0].length;

    memo = new int[m][n];
    int ans = 0;

    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            ans = Math.max(ans, dfs(matrix, i, j));
        }
    }

    return ans;
}

private int dfs(int[][] matrix, int i, int j) {
    if (memo[i][j] != 0) {
        return memo[i][j];
    }

    int best = 1;

    for (int[] d : dirs) {
        int x = i + d[0];
        int y = j + d[1];

        if (x < 0 || x >= matrix.length || y < 0 || y >= matrix[0].length) {
            continue;
        }

        if (matrix[x][y] > matrix[i][j]) {
            best = Math.max(best, 1 + dfs(matrix, x, y));
        }
    }

    memo[i][j] = best;
    return best;
}
```

### 一句话记忆

DFS 有重复子问题时，加 memo 就变成记忆化 DP。

---

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 子树状态重复算 | 记忆化 |
| 路径和贡献混淆 | 分清“返回值”和“全局答案” |
| 全局答案漏更新 | 用全局变量 |
| 图上 DP 出现环 | 必须确认 DAG 或用 visited 状态 |
| 父子选择约束 | 返回选 / 不选两个状态 |

---

## 面试速记

- 树形 DP 本质是后序 DFS。
- 每个节点向父节点返回信息。
- 返回值服务父节点，全局答案可以单独维护。
- 图上 DP 通常配合拓扑或记忆化。
