## 高频模板 1：最长回文子序列

### 秒判关键词

最长回文子序列、从两端收缩。

### 母题

LC516 Longest Palindromic Subsequence

### 标准代码

```java
public int longestPalindromeSubseq(String s) {
    int n = s.length();
    int[][] dp = new int[n][n];

    for (int i = n - 1; i >= 0; i--) {
        dp[i][i] = 1;

        for (int j = i + 1; j < n; j++) {
            if (s.charAt(i) == s.charAt(j)) {
                dp[i][j] = dp[i + 1][j - 1] + 2;
            } else {
                dp[i][j] = Math.max(dp[i + 1][j], dp[i][j - 1]);
            }
        }
    }

    return dp[0][n - 1];
}
```

### 一句话记忆

两端相等就一起用，不等就丢一端。

### 常见坑点

- 子序列不要求连续，子串要求连续。
- `i` 通常从后往前，保证 `dp[i+1][j]` 已经算过。

## 高频模板 2：戳气球

### 秒判关键词

删除一个元素会影响左右邻居，枚举最后一个操作。

### 母题

LC312 Burst Balloons

### 标准代码

```java
public int maxCoins(int[] nums) {
    int n = nums.length;
    int[] arr = new int[n + 2];

    arr[0] = 1;
    arr[n + 1] = 1;

    for (int i = 0; i < n; i++) {
        arr[i + 1] = nums[i];
    }

    int[][] dp = new int[n + 2][n + 2];

    for (int len = 2; len <= n + 1; len++) {
        for (int left = 0; left + len <= n + 1; left++) {
            int right = left + len;

            for (int k = left + 1; k < right; k++) {
                dp[left][right] = Math.max(
                    dp[left][right],
                    dp[left][k] + dp[k][right] + arr[left] * arr[k] * arr[right]
                );
            }
        }
    }

    return dp[0][n + 1];
}
```

### 一句话记忆

正着想会变邻居，反着想枚举最后一个被戳的气球。

## 高频模板 3：预测赢家

### 秒判关键词

两人轮流从两端取数，判断先手是否不输。

### 母题

LC486 Predict the Winner

### 标准代码

```java
public boolean predictTheWinner(int[] nums) {
    int n = nums.length;
    int[][] dp = new int[n][n];

    for (int i = 0; i < n; i++) {
        dp[i][i] = nums[i];
    }

    for (int len = 2; len <= n; len++) {
        for (int i = 0; i + len - 1 < n; i++) {
            int j = i + len - 1;

            dp[i][j] = Math.max(
                nums[i] - dp[i + 1][j],
                nums[j] - dp[i][j - 1]
            );
        }
    }

    return dp[0][n - 1] >= 0;
}
```

### 一句话记忆

记录当前玩家相对对手的最大分差。

## 高频模板 4：区间枚举通用骨架

### 标准代码

```java
for (int len = 2; len <= n; len++) {
    for (int i = 0; i + len - 1 < n; i++) {
        int j = i + len - 1;

        for (int k = i; k < j; k++) {
            dp[i][j] = combine(dp[i][j], dp[i][k], dp[k + 1][j]);
        }
    }
}
```

### 一句话记忆

先小区间，再大区间；必要时枚举切分点。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 遍历方向错 | 先算小区间 |
| 区间边界混乱 | 统一闭区间或开区间 |
| 回文子串和子序列混淆 | 子串连续，子序列可跳 |
| 戳气球正向模拟 | 改成枚举最后一个 |
| 博弈题直接求总分 | 记录分差更清晰 |

---

## 面试速记

- 区间 DP 核心是枚举区间长度。
- 博弈 DP 通常记录“当前玩家优势”。
- 从小区间推大区间。
- 删除类区间题常常反向想：枚举最后一步。
