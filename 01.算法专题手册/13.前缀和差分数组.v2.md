# 前缀和 / 差分数组

## 本章目标

训练区间题第一反应：看到频繁区间求和想到前缀和，看到频繁区间加法想到差分。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 多次查询区间和 | 前缀和 |
| 连续子数组和为 K | 前缀和 + HashMap |
| 二维矩阵区域和 | 二维前缀和 |
| 多次区间加法 | 差分数组 |
| 航班预订 / 区间增量 | 差分 |
| 有负数还问子数组和 | 前缀和，不用滑窗 |

## 高频模板 1：一维前缀和

### 秒判关键词

区间和、多次查询、`sum(l, r)`。

### 母题

LC303 Range Sum Query - Immutable

### 标准代码

```java
class NumArray {
    private int[] pre;

    public NumArray(int[] nums) {
        pre = new int[nums.length + 1];
        for (int i = 0; i < nums.length; i++) {
            pre[i + 1] = pre[i] + nums[i];
        }
    }

    public int sumRange(int left, int right) {
        return pre[right + 1] - pre[left];
    }
}
```

### 一句话记忆

`pre[i]` 表示前 `i` 个数的和，区间和用右减左。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Pivot Index | 左和等于右和 |
| Product Except Self | 前后缀乘积 |
| Running Sum | 原地累加即可 |

### 常见坑点

- `pre` 长度开 `n + 1` 最稳。
- 查询 `[l, r]` 是 `pre[r + 1] - pre[l]`。

## 高频模板 2：前缀和 + HashMap

### 秒判关键词

连续子数组、和为 K、统计个数、有负数。

### 母题

LC560 Subarray Sum Equals K

### 标准代码

```java
public int subarraySum(int[] nums, int k) {
    Map<Integer, Integer> count = new HashMap<>();
    count.put(0, 1);

    int sum = 0;
    int ans = 0;
    for (int x : nums) {
        sum += x;
        ans += count.getOrDefault(sum - k, 0);
        count.put(sum, count.getOrDefault(sum, 0) + 1);
    }
    return ans;
}
```

### 一句话记忆

当前是 `sum`，找之前出现过多少个 `sum - k`。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Subarray Sums Divisible by K | 存前缀和余数频次 |
| Continuous Subarray Sum | 余数相同且长度至少 2 |
| Contiguous Array | 0 当 -1，找和为 0 |

### 常见坑点

- `count.put(0, 1)` 不能漏。
- 先查再放，避免把当前前缀和当成之前的。
- 有负数时普通滑窗不稳定。

## 高频模板 3：二维前缀和

### 秒判关键词

矩阵区域和、左上到右下、多次查询。

### 母题

LC304 Range Sum Query 2D

### 标准代码

```java
class NumMatrix {
    private int[][] pre;

    public NumMatrix(int[][] matrix) {
        int m = matrix.length, n = matrix[0].length;
        pre = new int[m + 1][n + 1];
        for (int r = 0; r < m; r++) {
            for (int c = 0; c < n; c++) {
                pre[r + 1][c + 1] = pre[r][c + 1] + pre[r + 1][c]
                    - pre[r][c] + matrix[r][c];
            }
        }
    }

    public int sumRegion(int r1, int c1, int r2, int c2) {
        return pre[r2 + 1][c2 + 1] - pre[r1][c2 + 1]
            - pre[r2 + 1][c1] + pre[r1][c1];
    }
}
```

### 一句话记忆

二维区间和 = 大矩形 - 上面 - 左边 + 左上角。

### 高频变体

- Maximal Square 可用 DP，不是前缀和首选。
- Count Submatrices：常结合压缩行 + 一维前缀。

### 常见坑点

- 多减了一次左上角，要加回来。
- `pre` 用 `m + 1`、`n + 1` 避免边界特判。

## 高频模板 4：差分数组

### 秒判关键词

区间加、批量更新、最后一次性还原。

### 母题

LC1109 Corporate Flight Bookings

### 标准代码

```java
public int[] corpFlightBookings(int[][] bookings, int n) {
    int[] diff = new int[n + 1];

    for (int[] b : bookings) {
        int left = b[0] - 1;
        int right = b[1] - 1;
        int seats = b[2];
        diff[left] += seats;
        diff[right + 1] -= seats;
    }

    int[] ans = new int[n];
    int cur = 0;
    for (int i = 0; i < n; i++) {
        cur += diff[i];
        ans[i] = cur;
    }
    return ans;
}
```

### 一句话记忆

区间 `[l, r]` 加 x：`diff[l] += x`，`diff[r + 1] -= x`。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Car Pooling | 差分后检查容量是否超 |
| Range Addition | 标准区间加 |
| Meeting Rooms 类时间线 | 时间点差分 / 扫描线 |

### 常见坑点

- `right + 1` 要能访问，所以 diff 多开一格。
- 题目下标可能从 1 开始，要转成 0 开始。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 前缀和下标混乱 | 用 `n + 1` 写法 |
| 子数组和漏从 0 开始 | 初始化 `0 -> 1` |
| 有负数套滑窗 | 改前缀和 + Map |
| 二维前缀漏加左上 | 减两块后加回来 |
| 差分越界 | 多开一格 |

## 一题多变体

```text
Range Sum Query
↓
Subarray Sum Equals K
↓
Subarray Sums Divisible by K
```

```text
Corporate Flight Bookings
↓
Car Pooling
↓
Range Addition
```

## 面试速记

- 查询区间和：前缀和。
- 统计和为 K：前缀和 + Map。
- 矩阵区域和：二维前缀。
- 区间批量加：差分。
- 前缀和是“求区间”，差分是“改区间”。
