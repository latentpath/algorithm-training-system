# 字符串匹配基础 KMP

## 本章目标

训练字符串匹配选型：普通子串查找先会双指针 / API；面试要求线性匹配时，再想到 KMP 的 `next` 数组。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 判断是否包含子串 | 双指针 / `indexOf` 思路 |
| 找模式串第一次出现 | KMP 模板 |
| 大量重复前后缀 | KMP |
| 字符串周期 | KMP next 数组 |
| 多模式匹配 | Trie / Aho-Corasick，不是 KMP |
| 哈希匹配 | Rabin-Karp |

## 高频模板 1：KMP 子串查找

### 秒判关键词

`strStr`、模式串、线性匹配、不能回退主串。

### 母题

LC28 Find the Index of the First Occurrence in a String

### 标准代码

```java
public int strStr(String haystack, String needle) {
    if (needle.length() == 0) return 0;
    int[] lps = buildLps(needle);

    int j = 0;
    for (int i = 0; i < haystack.length(); i++) {
        while (j > 0 && haystack.charAt(i) != needle.charAt(j)) {
            j = lps[j - 1];
        }
        if (haystack.charAt(i) == needle.charAt(j)) {
            j++;
        }
        if (j == needle.length()) {
            return i - needle.length() + 1;
        }
    }
    return -1;
}

private int[] buildLps(String p) {
    int[] lps = new int[p.length()];
    int len = 0;
    for (int i = 1; i < p.length(); i++) {
        while (len > 0 && p.charAt(i) != p.charAt(len)) {
            len = lps[len - 1];
        }
        if (p.charAt(i) == p.charAt(len)) {
            len++;
        }
        lps[i] = len;
    }
    return lps;
}
```

### 一句话记忆

主串不回退，模式串按 `lps` 跳到还能继续匹配的位置。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Repeated Substring Pattern | 用 lps 判断周期 |
| Shortest Palindrome | 构造新串后跑 KMP |
| Find All Occurrences | 匹配后继续 `j = lps[j - 1]` |

### 常见坑点

- `lps[i]` 是 `p[0..i]` 的最长相等前后缀长度。
- mismatch 时跳 `lps[j - 1]`，不是 `lps[j]`。

## 高频模板 2：构建 LPS

### 秒判关键词

最长相等前后缀、next 数组、模式串自匹配。

### 母题

为模式串 `"ababaca"` 构建 lps。

### 标准代码片段

```java
private int[] buildLps(String p) {
    int[] lps = new int[p.length()];
    int len = 0;
    for (int i = 1; i < p.length(); i++) {
        while (len > 0 && p.charAt(i) != p.charAt(len)) {
            len = lps[len - 1];
        }
        if (p.charAt(i) == p.charAt(len)) {
            len++;
        }
        lps[i] = len;
    }
    return lps;
}
```

### 一句话记忆

`len` 表示当前最长前后缀长度，失配就继续跳短一点的前后缀。

### 高频变体

- 用 `next` 命名也可以，但含义要固定。
- 有些模板使用 `-1` 开头，不建议初学混用。

### 常见坑点

- `i` 从 1 开始，因为单字符前后缀长度为 0。
- while 跳转后仍要再判断一次是否相等。

## 高频模板 3：重复子串判断

### 秒判关键词

字符串是否由某个子串重复组成、周期。

### 母题

LC459 Repeated Substring Pattern

### 标准代码

```java
public boolean repeatedSubstringPattern(String s) {
    int n = s.length();
    int[] lps = buildLps(s);
    int len = lps[n - 1];
    return len > 0 && n % (n - len) == 0;
}
```

### 一句话记忆

最长前后缀留下的剩余长度，就是可能的最小周期。

### 高频变体

- 也可用 `(s + s).substring(1, 2n - 1).contains(s)` 判断。
- 面试要求解释时，KMP 版本更稳。

### 常见坑点

- `len == 0` 说明没有周期。
- 必须满足 `n % (n - len) == 0`。

## 高频模板 4：Rabin-Karp 了解级

### 秒判关键词

滚动哈希、子串快速比较、可能哈希冲突。

### 母题

重复 DNA 序列 / 字符串哈希匹配

### 标准思路

```text
1. 给每个字符映射数字。
2. 维护长度为 m 的窗口哈希。
3. 窗口右移时删左加右。
4. 哈希相等时，必要时再比较原字符串防冲突。
```

### 一句话记忆

Rabin-Karp 是给子串做滚动指纹。

### 高频变体

- Repeated DNA Sequences：固定长度 10 的滚动哈希。
- Longest Duplicate Substring：二分长度 + rolling hash。

### 常见坑点

- 哈希可能冲突。
- Java `substring` 大量创建字符串可能增加开销。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| KMP 失配时主串回退 | 主串 i 不回退，只跳 j |
| 跳错位置 | `j = lps[j - 1]` |
| 空模式串 | 返回 0 |
| 找所有匹配只返回第一次 | 匹配后继续跳 `lps[j-1]` |
| 周期判断只看 lps > 0 | 还要整除 |

## 一题多变体

```text
Find First Occurrence
↓
Find All Occurrences
↓
Repeated Substring Pattern
```

```text
Repeated DNA Sequences
↓
Rabin-Karp
↓
Longest Duplicate Substring
```

## 面试速记

- KMP 用于单模式串匹配。
- `lps` 是最长相等前后缀长度。
- 主串不回退，模式串回跳。
- 失配跳 `lps[j - 1]`。
- 周期题看 `n - lps[n-1]`。
