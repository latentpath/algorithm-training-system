# 栈与队列

## 本章目标

训练“最近未匹配”和“按层推进”的条件反射：括号、回退、层序、最短步数先想到栈或队列。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 括号匹配 | 栈 |
| 最近的上一个元素 | 栈 |
| 撤销 / 回退 / 路径还原 | 栈 |
| 层序遍历 | 队列 |
| 无权图最短步数 | BFS 队列 |
| 先进先出模拟 | 队列 |
| 需要队列最大值 | 单调队列 |

## 高频模板 1：括号匹配栈

### 秒判关键词

括号、匹配、有效、嵌套。

### 母题

LC20 Valid Parentheses

### 标准代码

```java
public boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(') stack.push(')');
        else if (c == '[') stack.push(']');
        else if (c == '{') stack.push('}');
        else {
            if (stack.isEmpty() || stack.pop() != c) {
                return false;
            }
        }
    }
    return stack.isEmpty();
}
```

### 一句话记忆

左括号入栈“期待的右括号”，右括号必须匹配栈顶。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Min Stack | 额外栈维护当前最小值 |
| Decode String | 栈保存之前的字符串和次数 |
| Simplify Path | 栈保存有效目录 |

### 常见坑点

- 遇到右括号时先判空。
- 最后栈必须为空。
- Java 用 `Deque`，不要用旧的 `Stack`。

## 高频模板 2：队列 BFS 层序

### 秒判关键词

层序、最短步数、从起点一圈圈扩散。

### 母题

二叉树层序遍历

### 标准代码

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> ans = new ArrayList<>();
    if (root == null) return ans;

    Queue<TreeNode> queue = new ArrayDeque<>();
    queue.offer(root);
    while (!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        ans.add(level);
    }
    return ans;
}
```

### 一句话记忆

队列按层弹出，`size` 锁住当前层。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Binary Tree Level Order | 每层收集 |
| Rotting Oranges | 多源 BFS，初始坏橘子都入队 |
| Word Ladder | BFS 求最短转换次数 |

### 常见坑点

- 层序题先保存 `size`，不要边加边当本层处理。
- BFS 最短路要及时标记 visited，通常入队时标记。

## 高频模板 3：单调栈

### 秒判关键词

下一个更大、下一个更小、右边第一个、温度等几天。

### 母题

LC739 Daily Temperatures

### 标准代码

```java
public int[] dailyTemperatures(int[] temperatures) {
    int n = temperatures.length;
    int[] ans = new int[n];
    Deque<Integer> stack = new ArrayDeque<>();

    for (int i = 0; i < n; i++) {
        while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) {
            int j = stack.pop();
            ans[j] = i - j;
        }
        stack.push(i);
    }
    return ans;
}
```

### 一句话记忆

栈里放“还没找到答案的下标”。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Next Greater Element | 用值映射答案 |
| Circular Next Greater | 遍历 `2n`，下标取模 |
| Largest Rectangle | 弹出时计算以被弹柱子为高的面积 |

### 常见坑点

- 栈里通常存下标，不是值。
- `while` 不是 `if`，当前元素可能解决多个旧元素。
- 循环数组只在前 `n` 次入栈。

## 高频模板 4：单调队列

### 秒判关键词

滑动窗口最大值、固定窗口极值、队首永远是答案。

### 母题

LC239 Sliding Window Maximum

### 标准代码

```java
public int[] maxSlidingWindow(int[] nums, int k) {
    int n = nums.length;
    int[] ans = new int[n - k + 1];
    Deque<Integer> deque = new ArrayDeque<>();

    for (int i = 0; i < n; i++) {
        while (!deque.isEmpty() && deque.peekFirst() <= i - k) {
            deque.pollFirst();
        }
        while (!deque.isEmpty() && nums[deque.peekLast()] <= nums[i]) {
            deque.pollLast();
        }
        deque.offerLast(i);
        if (i >= k - 1) {
            ans[i - k + 1] = nums[deque.peekFirst()];
        }
    }
    return ans;
}
```

### 一句话记忆

队首是窗口最大值，队尾淘汰更弱候选。

### 高频变体

- 窗口最小值：比较方向反过来。
- 带限制的子数组：维护最大队列和最小队列。

### 常见坑点

- 过期元素从队首弹。
- 更弱元素从队尾弹。
- 队列里存下标，才能判断是否过期。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 右括号时栈空 | 直接 false |
| BFS visited 标记太晚 | 入队时标记 |
| 层序漏 `size` | 每层先固定长度 |
| 单调栈用 `if` | 要用 `while` 连续弹 |
| 单调队列存值 | 存下标 |

## 面试速记

- 栈：最近未匹配。
- 队列：按顺序扩散。
- 单调栈：下一个更大 / 更小。
- 单调队列：窗口最值。
- BFS 最短路：队列 + visited。
