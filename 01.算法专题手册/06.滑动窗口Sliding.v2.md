# 滑动窗口 Sliding Window

## 本章目标

训练“连续子数组 / 子串”的第一反应：看到连续、最长、最短、窗口内满足条件，快速想到滑动窗口。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 连续子数组 / 子串 | 滑动窗口 |
| 最长且满足条件 | 可变窗口 |
| 最短覆盖 | 可变窗口 |
| 固定长度 k | 固定窗口 |
| 窗口内无重复 | Set / Map |
| 窗口内字符频次 | HashMap / 数组计数 |
| 数组含负数 + 和为 K | 不用滑窗，改前缀和 |

## 高频模板 1：最长合法窗口

### 秒判关键词

最长、无重复、至多 K 个、窗口变大变小。

### 母题

LC3 Longest Substring Without Repeating Characters

### 标准代码

```java
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> last = new HashMap<>();
    int left = 0;
    int ans = 0;

    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (last.containsKey(c)) {
            left = Math.max(left, last.get(c) + 1);
        }
        last.put(c, right);
        ans = Math.max(ans, right - left + 1);
    }
    return ans;
}
```

### 一句话记忆

右边扩，冲突时左边跳到冲突位置后面。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Longest Repeating Character Replacement | 维护窗口最大频次 |
| Max Consecutive Ones III | 至多 K 个 0 |
| Fruit Into Baskets | 至多两种字符 |

### 常见坑点

- `left` 不能往回退，所以要 `Math.max`。
- “最长”通常在每次合法后更新答案。

## 高频模板 2：最短覆盖窗口

### 秒判关键词

最短、覆盖、包含所有字符、窗口满足后收缩。

### 母题

LC76 Minimum Window Substring

### 标准代码

```java
public String minWindow(String s, String t) {
    Map<Character, Integer> need = new HashMap<>();
    for (char c : t.toCharArray()) {
        need.put(c, need.getOrDefault(c, 0) + 1);
    }

    Map<Character, Integer> window = new HashMap<>();
    int left = 0, valid = 0;
    int start = 0, len = Integer.MAX_VALUE;

    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (need.containsKey(c)) {
            window.put(c, window.getOrDefault(c, 0) + 1);
            if (window.get(c).equals(need.get(c))) valid++;
        }

        while (valid == need.size()) {
            if (right - left + 1 < len) {
                start = left;
                len = right - left + 1;
            }
            char d = s.charAt(left++);
            if (need.containsKey(d)) {
                if (window.get(d).equals(need.get(d))) valid--;
                window.put(d, window.get(d) - 1);
            }
        }
    }
    return len == Integer.MAX_VALUE ? "" : s.substring(start, start + len);
}
```

### 一句话记忆

先扩到满足，再尽量缩短。

### 高频变体

- Permutation in String：固定长度，可用计数窗口。
- Find All Anagrams：满足时记录起点。

### 常见坑点

- `valid` 表示达标的字符种类数，不是字符总数。
- 收缩窗口时，先判断是否破坏条件，再减计数。

## 高频模板 3：固定长度窗口

### 秒判关键词

长度为 k、每个长度 k 的子数组、平均值最大。

### 母题

LC643 Maximum Average Subarray I

### 标准代码

```java
public double findMaxAverage(int[] nums, int k) {
    int sum = 0;
    for (int i = 0; i < k; i++) {
        sum += nums[i];
    }
    int best = sum;

    for (int right = k; right < nums.length; right++) {
        sum += nums[right];
        sum -= nums[right - k];
        best = Math.max(best, sum);
    }
    return best * 1.0 / k;
}
```

### 一句话记忆

进一个，出一个，窗口长度永远是 k。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Find All Anagrams | 固定长度为 `p.length()` |
| Permutation in String | 固定长度比较计数 |
| Sliding Window Maximum | 固定窗口 + 单调队列 |

### 常见坑点

- 固定窗口不需要 while 收缩。
- 初始化前 k 个，再从 k 开始滑。

## 高频模板 4：窗口计数型

### 秒判关键词

至多 K 种、字符频次、窗口内不同元素数。

### 母题

LC904 Fruit Into Baskets

### 标准代码

```java
public int totalFruit(int[] fruits) {
    Map<Integer, Integer> count = new HashMap<>();
    int left = 0, ans = 0;

    for (int right = 0; right < fruits.length; right++) {
        count.put(fruits[right], count.getOrDefault(fruits[right], 0) + 1);

        while (count.size() > 2) {
            int x = fruits[left++];
            count.put(x, count.get(x) - 1);
            if (count.get(x) == 0) count.remove(x);
        }
        ans = Math.max(ans, right - left + 1);
    }
    return ans;
}
```

### 一句话记忆

Map 维护窗口内计数，超过限制就缩左边。

### 高频变体

- Longest Substring with At Most K Distinct Characters：限制种类数为 k。
- Max Consecutive Ones III：计数 0 的数量。

### 常见坑点

- 计数减到 0 要删除 key。
- 窗口是否合法由 Map 状态决定。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 非连续问题套滑窗 | 滑窗只管连续 |
| 有负数求和为 K | 前缀和 + HashMap |
| 最短窗口忘收缩 | 满足后 while 缩 |
| 固定窗口写 while | 进一个出一个 |
| Map 计数 0 不删 | `size()` 会错 |

## 一题多变体

```text
Longest Substring Without Repeating
↓
Fruit Into Baskets
↓
Max Consecutive Ones III
```

```text
Minimum Window Substring
↓
Permutation in String
↓
Find All Anagrams
```

```text
Maximum Average Subarray
↓
Sliding Window Maximum
```

## 面试速记

- 滑窗只看连续。
- 最长：扩右，非法缩左，合法更新。
- 最短：扩右，满足后缩左，缩前更新。
- 固定 k：进一个出一个。
- 有负数求和：优先前缀和。
