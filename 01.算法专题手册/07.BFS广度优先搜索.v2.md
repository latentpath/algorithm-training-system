# BFS 广度优先搜索

## 本章目标

训练“按层扩散”的第一反应：看到最短步数、层序、多源扩散，优先想到 BFS 队列。

## 秒判卡

| 看到什么 | 直接想到 |
|---|---|
| 无权图最短路径 | BFS |
| 最少步数 / 最少操作 | BFS |
| 二叉树层序 | 队列 BFS |
| 从多个起点同时扩散 | 多源 BFS |
| 网格最近距离 | BFS |
| 带权最短路 | Dijkstra，不是普通 BFS |

## 高频模板 1：标准 BFS 最短步数

### 秒判关键词

最短、最少步数、无权图、每一步代价相同。

### 母题

LC111 Minimum Depth of Binary Tree

### 标准代码

```java
public int minDepth(TreeNode root) {
    if (root == null) return 0;

    Queue<TreeNode> queue = new ArrayDeque<>();
    queue.offer(root);
    int depth = 1;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            if (node.left == null && node.right == null) {
                return depth;
            }
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        depth++;
    }
    return depth;
}
```

### 一句话记忆

BFS 第一层遇到目标，就是最短答案。

### 高频变体

| 题目 | 变化点 |
|---|---|
| Word Ladder | 字符串作为节点 |
| Open the Lock | 状态作为节点 |
| Shortest Path in Binary Matrix | 网格 8 方向 |

### 常见坑点

- BFS 求最短路时，入队时就标记 visited。
- 层数可以用 `depth`，也可以把距离随节点入队。

## 高频模板 2：网格 BFS

### 秒判关键词

网格、上下左右、最短路径、障碍物。

### 母题

网格从起点到终点的最短步数。

### 标准代码

```java
public int shortestPath(int[][] grid) {
    int m = grid.length, n = grid[0].length;
    if (grid[0][0] == 1) return -1;

    int[][] dirs = {{1,0},{-1,0},{0,1},{0,-1}};
    boolean[][] visited = new boolean[m][n];
    Queue<int[]> queue = new ArrayDeque<>();
    queue.offer(new int[]{0, 0});
    visited[0][0] = true;
    int step = 0;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int[] cur = queue.poll();
            int r = cur[0], c = cur[1];
            if (r == m - 1 && c == n - 1) return step;

            for (int[] d : dirs) {
                int nr = r + d[0], nc = c + d[1];
                if (nr < 0 || nr >= m || nc < 0 || nc >= n) continue;
                if (grid[nr][nc] == 1 || visited[nr][nc]) continue;
                visited[nr][nc] = true;
                queue.offer(new int[]{nr, nc});
            }
        }
        step++;
    }
    return -1;
}
```

### 一句话记忆

队列存坐标，按层扩散，第一次到终点就是最短。

### 高频变体

- 8 方向：扩展 `dirs`。
- 可以消除障碍：状态要带剩余次数。
- 多把钥匙 / 多状态：visited 变三维或状态压缩。

### 常见坑点

- visited 只用坐标不够时，要把“状态”也放进去。
- `step` 从 0 还是 1 取决于题目定义边数还是点数。

## 高频模板 3：多源 BFS

### 秒判关键词

多个起点、同时扩散、最近距离、腐烂橘子。

### 母题

LC994 Rotting Oranges

### 标准代码

```java
public int orangesRotting(int[][] grid) {
    int m = grid.length, n = grid[0].length;
    Queue<int[]> queue = new ArrayDeque<>();
    int fresh = 0;

    for (int r = 0; r < m; r++) {
        for (int c = 0; c < n; c++) {
            if (grid[r][c] == 2) queue.offer(new int[]{r, c});
            if (grid[r][c] == 1) fresh++;
        }
    }

    int[][] dirs = {{1,0},{-1,0},{0,1},{0,-1}};
    int minutes = 0;
    while (!queue.isEmpty() && fresh > 0) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int[] cur = queue.poll();
            for (int[] d : dirs) {
                int nr = cur[0] + d[0], nc = cur[1] + d[1];
                if (nr < 0 || nr >= m || nc < 0 || nc >= n) continue;
                if (grid[nr][nc] != 1) continue;
                grid[nr][nc] = 2;
                fresh--;
                queue.offer(new int[]{nr, nc});
            }
        }
        minutes++;
    }
    return fresh == 0 ? minutes : -1;
}
```

### 一句话记忆

所有源点先入队，一起向外扩散。

### 高频变体

| 题目 | 变化点 |
|---|---|
| 01 Matrix | 所有 0 作为源点 |
| Walls and Gates | 所有门作为源点 |
| As Far from Land as Possible | 所有陆地作为源点 |

### 常见坑点

- 多源不是对每个起点跑一次 BFS。
- 初始所有源点同时入队，层数才正确。

## 高频模板 4：状态 BFS

### 秒判关键词

每步变换一个状态、最少操作次数、状态空间有限。

### 母题

LC752 Open the Lock

### 标准代码

```java
public int openLock(String[] deadends, String target) {
    Set<String> dead = new HashSet<>(Arrays.asList(deadends));
    if (dead.contains("0000")) return -1;

    Queue<String> queue = new ArrayDeque<>();
    Set<String> visited = new HashSet<>();
    queue.offer("0000");
    visited.add("0000");
    int step = 0;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            String cur = queue.poll();
            if (cur.equals(target)) return step;
            for (String next : nextStates(cur)) {
                if (dead.contains(next) || visited.contains(next)) continue;
                visited.add(next);
                queue.offer(next);
            }
        }
        step++;
    }
    return -1;
}
```

### 一句话记忆

把每个局面当节点，把一次操作当边。

### 高频变体

- Word Ladder：每次改一个字符。
- Sliding Puzzle：每次交换空格。

### 常见坑点

- 状态要能唯一表示。
- 状态生成函数不要漏情况。

## 高频坑点

| 坑点 | 正确反应 |
|---|---|
| 入队后不标记 | 会重复入队爆炸 |
| 多源 BFS 跑多次 | 所有源点一起入队 |
| 带权图用 BFS | 改 Dijkstra |
| 状态 visited 维度不够 | 把关键状态也纳入 key |
| 层数 off-by-one | 明确答案是边数还是节点数 |

## 一题多变体

```text
Minimum Depth
↓
Word Ladder
↓
Open the Lock
```

```text
Rotting Oranges
↓
01 Matrix
↓
Walls and Gates
```

## 面试速记

- BFS：队列 + visited + 按层。
- 无权最短路：BFS。
- 多源：所有起点先入队。
- 状态题：状态是节点，操作是边。
- 入队时标记 visited。
